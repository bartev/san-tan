The Top Five packages, used on arXiv.org\footnote{according to the arXMLiv project\label{fn:project}}:
\begin{enumerate}
  \item graphicx
  \item amssymb \label{item:amssymb}
  \item amsmath \label{item:amsmath}
  \item epsfig \label{item:epsfig}
  \item amsfonts
\end{enumerate}
